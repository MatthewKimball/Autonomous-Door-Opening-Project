-------Door Opening Project Files-------

- Custom ROSJava Messages:
This is the custom .jar message file need to be copied into the Matlab message library to allow its use.

- Matlab Code:
The code within this folder is the Matlab functions written to interface the vision detection code and the ROS based manipulation code

- ROS Integrated Files:
This the code that is built ontop of Kinova's offical ROS files, the manipulation routines are embedded within the main command node.

- Simulation Files:
This folder contains a working simulation model of the Mico arm and Guaibot, Moveit! can control the Gazebo model of the Arm.
